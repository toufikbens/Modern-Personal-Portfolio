# Modern Personal Portfolio

A modern dark-themed personal portfolio website built using HTML, CSS, and JavaScript.
This project focuses on responsive design, smooth animations, clean UI structure, and a modern tech-inspired aesthetic.

## Features

* Responsive layout for desktop and mobile
* Modern dark tech UI
* Smooth scrolling and animations
* Interactive project cards
* Fully static website
* Clean and organized code structure

## Technologies Used

* HTML5
* CSS3
* JavaScript
